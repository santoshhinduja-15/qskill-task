This repository contains two frontend internship tasks developed using **HTML, CSS, Bootstrap, and JavaScript**, focusing on clean UI design, user experience, form validation, and API integration.

## Task 1: Login Page
A responsive login page with a modern user interface.  
It includes email and password input fields with client-side validations, a password show/hide toggle, and user feedback using Bootstrap dismissible alerts.  
The task demonstrates form handling, validation logic, and basic JavaScript interactions.

## Key Features:
- Responsive design using Bootstrap  
- Email and password validation  
- Password show/hide toggle  
- Bootstrap dismissible alerts for feedback  

## Task 2: Spotify Song Downloader
A web application that allows users to paste a Spotify track URL and fetch song details using a **RapidAPI integration**.  
The application displays song information such as title, artist, album, and cover image, along with a downloadable audio link.  
It focuses on clean UI, proper error handling, and a smooth user experience.

## Key Features:
- Spotify track URL input  
- RapidAPI integration  
- Displays song details and cover image  
- Download link generation  
- Error handling and loading state  

## Technologies Used
- HTML  
- CSS  
- Bootstrap  
- JavaScript  
- RapidAPI  

## Purpose
These tasks were created as part of an internship assignment to demonstrate frontend development skills, UI/UX understanding, and practical implementation of JavaScript and third-party APIs.
